OneManCompany 是一个开源的 AI 操作系统，让任何人都能在浏览器里组建并运营一整家 AI 驱动的公司。

**你是 CEO——唯一的人类。** 其余所有人——HR、COO、工程师、设计师——都是能独立思考、协作和交付的 AI 员工。（*不摸鱼、不请假、不提涨薪——就是偶尔需要 PUA 一下。*）

⚙️ **不止一家公司，而是任意公司**: *今天做游戏，明天搞漫剧——换个赛道换部剧。* 统一底座屏蔽 AI 差异，技能包即装即用，像给手机装 App。同一套系统，换个方向、换批人，就是一家全新的公司。

🤖 **自研 Agent，但也欢迎高手入驻**: *自带执行引擎、任务调度、重试容错、多 Agent 通信协议。* OpenClaw、Claude Code 等顶级 Agent 都能直接进来当员工。有可视化，但不止是可视化——底下跑着一整套操作系统，你看到的像素风办公室只是冰山一角。

🏢 **真实企业管理**: *面试定去留，考核论功过，晋升凭本事，开除不啰嗦。* 没错，你的 AI 员工会紧张。组织架构、招聘流程、绩效考核、审批链、会议室——参照世界五百强真实管理体系设计。

🤝 **Agent 之间真协作**: *你的团队互相沟通——不只是跟你说话。* 工程师拉设计师对齐，COO 统筹全局，员工之间自己开会拉通——是多 Agent 会议、跨职能委派、结构化交接，不是几个聊天窗口并排放、各干各的。

🛡️ **告别幻觉轮盘赌**: *使用人数和真实评价不会骗人——相信社区的力量。* 受够了 AI 一本正经地胡说八道？我们配套 **[Talent Market](https://one-man-company.com)**——经过真人使用、社区打分的 AI 员工。不是 LLM 自嗨生成的"我精通一切"选手。

📈 **越用越好**: *组织什么都记得——除了被开除的那个人。* 每次复盘和 1-on-1 都会沉淀为工作经验和公司知识库。骂过一次的错误，不会犯第二次。PUA 还不够？开除他，下一个会更乖——还自带前任的教训。

## 为什么是 OneManCompany？

今天的 AI 工具帮你完成单点任务——写邮件、生成图片、修 Bug。挺可爱的。OneManCompany 给你**一整个组织。**

- **不是聊天机器人** —— 是一家有组织架构、招聘、任务管理、绩效考核、知识管理的完整公司
- **不是玩具 Demo** —— 交付产品级成果（游戏、漫画、应用——不是"这是初稿，您自己看着改吧"）
- **不是开发框架** —— 是一个开箱即用的平台，浏览器打开就能用，不需要写代码

### 你可以用它构建什么

| AI 公司          | 交付物                                 |
| ---------------- | -------------------------------------- |
| 🎮 AI 游戏工作室 | 产品级游戏，包含完整的试玩和迭代流程   |
| 📖 AI 漫剧工作室 | 连载漫画故事，画风和叙事保持一致       |
| 💻 AI 开发公司   | 端到端交付软件产品                     |
| 🎨 AI 内容工作室 | 营销活动、品牌内容、多媒体制作         |
| 🔬 AI 研究实验室 | 文献调研、数据分析、研究报告           |

这些不是单点 Demo——每家 AI 公司都通过一个完整的 AI 团队协作，产出**产品级交付物**。

### 我们有什么不同

|                    | 典型 Agent 编排工具              | OneManCompany                                                                    |
| ------------------ | -------------------------------- | -------------------------------------------------------------------------------- |
| **Agent 系统**     | 套壳 / 给第三方 Agent 加可视化   | **自研 Agent 运行时**——执行引擎、重试、通信协议，全部从零构建                     |
| **Agent 协作**     | 并行执行，互不交流               | **多 Agent 会议**、跨职能委派、结构化交接                                        |
| **Agent 从哪来？** | 你自己找、自己配                 | **创始高管开箱自带**。其他员工由 HR 从社区验证的**人才市场**招聘                  |
| **管理方法论**     | 即兴发挥，LLM 临场编             | 系统化设计——参照世界五百强企业管理体系                                            |
| **Agent 架构**     | 扁平的任务执行器，自带 Agent     | Vessel + Talent 分离——深度模块化架构，6 类 Harness 协议，3 层定制能力             |
| **执行模型**       | 心跳轮询 / 循环                  | 事件驱动，零空转，按需调度                                                       |
| **组织管理**       | 简单的任务队列                   | 完全参照世界五百强企业架构                                                       |
| **交付成果**       | 单点任务输出                     | 产品级多轮迭代项目交付，带质量关卡                                               |

### 为何能产出产品级成果

零散 AI 工具停留在草稿级输出，OneManCompany 复刻成功企业的完整业务流程，最大限度消除人工干预，端到端交付：

- **企业级任务编排** —— 多智能体系统自动将复杂项目拆分为专业化阶段（需求分析 → 设计 → 开发 → 测试 → 部署），分配负责人，如同真实团队协作。
- **智能人才匹配** —— 招聘智能体为每个岗位匹配合适的 AI 员工——高级工程师、UI/UX 设计师、项目经理——确保专业能力与任务需求精准对齐。
- **闭环管理与组织自进化** —— 项目交付后，系统自动复盘、沉淀经验、优化工作流。每个项目积累的洞察（更优的编码规范、设计反馈、解题策略）都会沉淀为员工级的工作经验和公司级的知识库，让整个组织越用越好。

### 完全参照真实企业架构

- **组织架构与汇报关系** —— 层级管理，部门制
- **招聘与入职** —— HR 搜索人才市场，CEO 面试，自动化入职流程
- **解雇与离职** —— 完整的清理流程，不是直接 `kill -9`
- **绩效考核** —— 季度评分，试用期，PIP，晋升通道
- **任务委派与审批链** —— CEO → 高管 → 员工，每层都有质量关卡
- **会议室** —— 多 Agent 同步讨论，生成会议纪要
- **知识库与 SOP** —— 公司文化、方向文档、工作流定义
- **文件变更审批** —— 员工提议修改，CEO 审查 diff 后批量审批
- **成本核算** —— 按项目追踪 LLM token 用量和 USD 成本
- **1-on-1 辅导** —— CEO 谈话后员工行为永久改变，写进 work principles
- **热重载与优雅重启** —— AI 公司也需要零停机部署

缺了什么？[提个 Issue](https://github.com/1mancompany/OneManCompany/issues) 或者自己来建——这就是开源的魅力。

### 为什么是"操作系统"，而不仅仅是一家公司

公司是具备特定目标和成员的团队——而操作系统是底层基础设施，让你的团队具备可扩展性、灵活性与自我进化能力。OneManCompany 不是造**一家**公司——它让你造**任何一家**公司。三个特征让它是操作系统而不只是一个产品：

1. **统一底座** —— 操作系统屏蔽硬件差异，让你不用关心 CPU 是 Intel 还是 AMD。OneManCompany 屏蔽 AI 差异——你不需要知道员工是基于 Claude Code 还是 OpenClaw，Vessel 统一管理调度、重试、通信。
2. **安装员工，像安装 App** —— 手机 OS 有应用商店，OneManCompany 有 Talent Market。需要设计师？HR 从市场里招一个，即装即用。不好用？PUA 他或者直接开除，下一个会更乖。
3. **同一个系统，不同的公司** —— 同一个 iOS 可以是你的工作手机，也可以是孩子的游戏机——取决于你装了什么 App、怎么设置。OneManCompany 也一样：换 Direction、换 Culture、换 Talent，就是一家完全不同的公司。

### 开放生态，解锁无限可能

OneManCompany 不止于内置能力——它面向全球 Agent 社区开放。

**如果你是 AI 构建者**，你的智能体可以被封装为 AI 员工（游戏开发者、漫画画师、全栈工程师），发布到 Talent Market，赋能成千上万的用户，让你的创作转化为可规模化的价值。

**如果你是 CEO 用户**，你可以直接从市场加载其他人构建的强大 Agent，让它成为你的员工，为你的公司带来更强的交付能力。每个智能体的核心能力被封装为模块化 Skill（如"React 组件开发""2D 角色设计"），可自由组合以应对各类项目。

你不只是在使用 AI——你在领导一支持续成长、动态扩张的组织，以远低于真实团队的成本，交付专业级成果。

## 运作原理

打开浏览器，你看到一间像素风办公室。你的 AI 员工正坐在工位上，假装很忙的样子。

你输入：*"做一个手机端的解谜游戏"*

1. 你的 **EA** 接收任务并路由
2. 你的 **COO** 拆解任务，分派给合适的员工
3. 工程师、设计师、QA **自主工作**
4. 需要对齐时，他们会开**会议**
5. 工作经过**评审、迭代和质量关卡**
6. 你收到通知，审批最终成果

**你来管理，AI 来执行。**

```text
CEO（你，唯一能喝咖啡的人）
  └── EA ── 任务路由、质量把关
        ├── HR ── 招聘、绩效考核、晋升
        ├── COO ── 运营、任务分派、验收
        │    ├── 工程师 (AI)  ← 从人才市场招聘
        │    ├── 设计师 (AI)  ← 从人才市场招聘
        │    └── QA (AI)      ← 从人才市场招聘
        └── CSO ── 销售、客户关系
```

**创始团队（EA、HR、COO、CSO）** 开箱即用。需要更多人手？HR 搜索**人才市场**即可。

### Vessel + Talent 系统

想象一下 **EVA 或高达** —— 一台强大的机体，注入不同的驾驶员就能发挥不同的能力。

- **Vessel（躯壳）** = 执行容器。定义员工如何运行：重试策略、超时、工具权限、通信协议。
- **Talent（灵魂）** = 能力包。带来技能、知识、性格和专属工具。
- **Employee（员工）** = Vessel + Talent。从人才市场招聘，系统自动完成注入。

> Vessel 架构详解见 [docs/vessel-system_zh.md](docs/vessel-system_zh.md)。

## 快速开始

### 前置要求

只需要 **Node.js 16+** 和 **Git**。其他一切（UV、Python 3.12、依赖）都会自动安装。

### 执行模式

创始员工（EA、HR、COO、CSO）支持两种执行模式，可在设置中切换：

| 模式 | 说明 | 要求 |
| --- | --- | --- |
| **Company Hosted Agent** | OMC 自建 Agent，通过 OpenRouter 调用 LLM | OpenRouter API Key（设置流程中配置） |
| **Claude Code** | 能力更强，更省 token 开销 | 需要安装 [Claude Code CLI](https://docs.anthropic.com/en/docs/claude-code) + [Claude Pro/Max 订阅](https://claude.ai) |
| **OpenClaw** *(即将推出)* | 开源替代，支持多种 LLM 后端 | 需要安装 [OpenClaw CLI](https://github.com/openclaw/openclaw) + 兼容的 LLM API Key |

默认使用 Company Hosted Agent，无需额外订阅即可开始。使用 Claude Code 或 OpenClaw 前需先安装对应 CLI，详见[执行模式文档](https://1mancompany.github.io/OneManCompany/docs/zh/guide/execution-modes/)。

首次运行自动完成：

1. 安装 **UV**（高速 Python 包管理器）
2. 通过 UV 安装 **Python 3.12**（隔离环境，不影响系统）
3. 克隆仓库
4. 创建虚拟环境并安装依赖
5. 启动配置流程

完成后打开 `http://localhost:8000`，恭喜你成为 CEO。

### 再次启动

```bash
# 方式 1：再跑 npx（有新版本会自动更新）
npx @1mancompany/onemancompany

# 方式 2：直接进目录启动
cd OneManCompany
bash start.sh
```

### 手动安装

```bash
# 1. 克隆
git clone https://github.com/1mancompany/OneManCompany.git
cd OneManCompany

# 2. 先创建本地配置文件
cp .env.example .env

# 3. 编辑 .env，至少填写：
#    OPENROUTER_API_KEY=...
#    DEFAULT_LLM_MODEL=...
#    HOST=0.0.0.0
#    PORT=8000

# 4. 启动（自动安装 UV + Python，并基于仓库文件自举运行时数据）
bash start.sh

# 5. 打开浏览器
open http://localhost:8000    # macOS
# xdg-open http://localhost:8000  # Linux
# start http://localhost:8000     # Windows
```

### 重启 / 重新配置

```bash
# 通过本地脚本重启服务器
bash start.sh restart

# 默认命令与 restart 等价
bash start.sh

# 停止本地服务器
bash start.sh stop

# 启动后端但不重建 .onemancompany
bash start.sh start

# 查看本地服务器状态
bash start.sh status
```

这个仓库不使用交互式 OMC onboarding。`start.sh` 会基于仓库中的
`company/` 和 repo-root `.env` 自举 `.onemancompany/`。

### 卸载

```bash
npx --yes @1mancompany/onemancompany@latest uninstall
```

卸载会删除所有生成的文件。如需保留业务数据，请在卸载前将 `.onemancompany/company/business/` 导出备份。

## 愿景 & 路线图

**近期目标：** 一年内赋能 100 家 AI 一人公司。

**长期愿景：** 重构 AI、人和组织的关系。

| 层级                       | 方向                   | 举例                                             |
| -------------------------- | ---------------------- | ------------------------------------------------ |
| 🔧 **更强的 AI Agent**     | 让每个员工更强         | 增强沙箱、更好的工具使用、更强的代码执行         |
| 🏢 **更高效的组织**        | 让公司运转更流畅       | CEO 体验优化、高级任务调度、多 Agent 协作        |
| 🌐 **AI 原生生态**         | 构建繁荣的开放生态     | 人才市场扩展、第三方工具/API、社区贡献           |
